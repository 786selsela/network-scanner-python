# 🔐 Network Scanner (Python + Nmap)

## 📌 Description
This is a basic network scanner built using Python and Nmap. It scans a target IP address and detects open ports.

## 🚀 Features
- Scan IP address
- Detect open ports
- Show port status (open, filtered)

## 🛠 Tools Used
- Python
- Nmap
- python-nmap library

## ▶️ How to Run
1. Install Nmap
2. Install python-nmap:
   pip install python-nmap
3. Run the script:
   python network_scanner.py

## ⚠️ Note
This tool is for educational purposes only. Only scan authorized systems.
